# 🏨 Hotel Booking Analysis & Customer Retention Dashboard #

## Project Overview:-
This project focuses on analyzing hotel booking data to understand customer behavior, cancellation patterns, and pricing trends.The goal is to reduce cancellations and improve revenue using data-driven insights.

## Business Objectives:-
* Reduce booking cancellations
* Optimize pricing strategy (ADR)
* Identify high-value customer segments
* Analyze seasonal booking trends

## Tools & Technologies Used:-
* Python (Pandas, NumPy)
* Data Visualization (Matplotlib, Seaborn)
* Power BI (Dashboard)
* Jupyter Notebook

## Key KPIs:-
* Total Bookings
* Total Revenue
* Cancellation Rate (%)
* Average Daily Rate (ADR)

## Key Insights:-

* High lead time increases cancellation risk.
* Higher ADR shows price sensitivity in customers.
* Corporate customers have lower cancellation rates.
* Strong seasonal trends affect bookings and pricing.
* Certain countries contribute most to bookings and cancellations.

## Data Processing Steps:-
* Data Cleaning (handling missing values, removing duplicates)
* Feature Engineering (Total Guests, Total Nights)
* Exploratory Data Analysis (EDA)
* Data Visualization

## Machine Learning Model:-
* Model Used: Logistic Regression
* Objective: Predict booking cancellations
* Evaluation Metrics: Accuracy, Precision, Recall

## Conclusion:-
This project provides valuable insights into customer behavior and helps businesses make better pricing and retention decisions.
